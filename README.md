# 🎨 Random Color Generator

A simple and interactive web application that generates random colors in **HEX** and **RGB** formats. Users can generate colors instantly and copy them with a single click.

## 📌 Features

- 🎨 Generate custom **RGB colors** using sliders
- 📋 Copy color codes to clipboard
- 🌈 Real-time UI color updates
